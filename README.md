# 👨‍💼 Employee Management System (Full Stack Java + React)

A full-stack web application for managing employees, built with **Java Spring Boot** (backend) and **React + Vite + Tailwind CSS** (frontend).  
This project demonstrates complete **CRUD operations** (Create, Read, Update, Delete) with a modern UI and REST API integration.

---

## 🚀 Features
- ➕ Add new employees with details (First Name, Last Name, Email).
- 📋 View a list of all employees in a styled dashboard.
- ✏️ Update employee information.
- ❌ Delete employees with confirmation.
- 🎨 Responsive UI built with **Tailwind CSS**.
- ⚡ RESTful APIs powered by **Spring Boot**.
- 🗄️ MySQL database integration.

---

## 🛠️ Tech Stack
### Backend
- Java 17
- Spring Boot
- Spring Data JPA
- MySQL
- Lombok

### Frontend
- React (Hooks, Router)
- Vite
- Tailwind CSS
- Axios
